# The following list of configurations was built into the Windows
# Brain Jam program (though not, of course, in this precise format).
# Peter Liepa wrote the original program and created a set of
# configurations. Bert van Oortmarssen later wrote a program to
# identify which configurations in the original set were unsolvable,
# which Peter Liepa then removed from later versions of his program.
# His program also determined the minimum move counts for the solvable
# configurations. They have explicitly given their permission for the
# list of configurations to be used here.
#
# Each line of this string corresponds to one configuration. The
# configuration ID begins the line, followed by 52 letters giving the
# sequence of cards in the deck. The layout is constructed from the
# deck by laying out cards from left to right, top to bottom. At the
# end of each line is the size of the shortest possible answer for
# that configuration.
#
# Each card in the deck is encoded by a single letter. The letters are
# assigned alphabetically, uppercase first, with ranks going from Ace
# to King, and suits in the order clubs, diamonds, hearts, spades.
# Thus A=Ac, B=Ad, C=Ah, D=As, E=2c, F=2d, and so on up to z=Ks.
